I am an aspiring Software Developer, QA Automation Engineer, and Frontend Developer passionate about building user-friendly applications and ensuring software quality.

## Features

- **Responsive Design**: Built with pure CSS to ensure a great viewing experience on all devices.
- **Dark Mode Aesthetic**: A sleek, modern dark UI using Google Fonts (Poppins) and FontAwesome icons.
- **Dynamic Typing Animation**: Included on the hero section for a lively first impression.
- **Skills Progress Bars**: Animated visual representation of my technical skills.
- **Project Showcase**: A dedicated section to highlight my most recent work.
- **Experience Timeline**: A clean timeline layout showing my past internships for easy reading.
- **Interactive Navigation**: Sticky header and smooth scrolling to seamlessly navigate between sections.

## Local Setup

To run this project locally on your machine, simply:
1. Clone this repository: `git clone https://github.com/NidyaB/Portfolio.git`
2. Open the directory: `cd Portfolio`
3. Just double-click `index.html` to open it in your browser, or use an extension like VS Code Live Server.
